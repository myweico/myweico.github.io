发表于 2020-02-21 更新于 2020-02-20 分类于 技巧

有时候发现，国内的 github 的图片显示不了，这就让人很苦恼。

其实是存放图片的avatars0.githubusercontent.com等相关网址被墙了原因。

只需要修改 host 文件的相关内容即可以让 github 正常显示图片。

host 文件位置

  • Mac OS: /etc/hosts
